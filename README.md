# Paragon Grid Studio

The official website for **Paragon Grid Studio, LLC** — a Houston-based web design studio building trust-first websites for local businesses with strong reviews and weak online presence.

🌐 **Live site:** _coming soon via Vercel_

## What's inside

```
.
├── index.html           # Homepage
├── privacy-policy.html  # Privacy Policy
├── terms-of-use.html    # Terms of Use
└── README.md
```

Single-file HTML + Tailwind (via CDN). No build step, no dependencies. Just open `index.html` in a browser.

## Stack

- **HTML5** — hand-coded, semantic
- **Tailwind CSS** — via CDN for zero-build simplicity
- **Google Fonts** — Fraunces (display), Instrument Sans (body), Instrument Serif (italic accents)
- **Vanilla JavaScript** — scroll reveals, form validation, smooth scroll

## Local development

Just open `index.html` in your browser. Or serve the folder with any static server:

```bash
# Python
python3 -m http.server 3000

# Node
npx serve .
```

Then visit `http://localhost:3000`.

## Deployment

Deployed via **Vercel**. Any push to `main` triggers an automatic production deploy.
